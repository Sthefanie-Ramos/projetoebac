# projetoebac
Projeto de calculadora simples do curso EBAC

Esse projeto é uma calculadora simples que fiz usando Python e também um script em .sh.


Como executar o arquivo .sh:
No WSL ou Linux, abrir o terminal na pasta do arquivo e usar:
chmod +x calculadora.sh  
./calculadora.sh  


Explicação do código em Python:
O código em Python funciona como uma calculadora. Ele pede pro usuário escolher uma operação (soma, subtração, multiplicação, divisão ou potência) e digitar os números. Também coloquei validação pra evitar erro, tipo divisão por zero e limite de números. Usei:
- input() pra receber os valores  
- if e elif pra decidir qual operação fazer  
- print() pra mostrar o resultado  
- while pra repetir o programa  


**Observação:
O script .sh faz operações básicas direto no terminal.
